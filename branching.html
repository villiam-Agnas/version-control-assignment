[Diffrent Git commands and when to use them]

  • Create and switch to a new branch with git checkout -b feature-branch. In our case, we first created a development branch from main. Then, for each separate page, we created feature branches. For example, this page was made on the branch git checkout -b development-branching.

  • If we want to switch back to our development branch we can use git checkout development This allows me to from the development-branching to development branch.
  • If we want to delete a branch, then we can use git branch -d development-branching This deletes the development-branching from the development branch.
