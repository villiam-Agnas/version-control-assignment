# Git Tutorial by Team 4 🎉

Welcome to our **Git Tutorial Website**!  
Here, we’ll help you get started with Git and understand the essentials of version control.

## **Overview**
This website covers the following core Git topics:
- **How to create projects** with Git  
- **Make changes** to a repository  
- Create **branches** for a better workflow  

## **Usage of this Website**
This website helps you learn the fundamentals of starting a GitHub project and collaborating efficiently with your team.

## **How to Contribute**
1. Fork the repo  
2. Create a new branch for your feature or fix  
3. Make changes and commit them  
4. Push the branch to your fork  
5. Open a pull request
