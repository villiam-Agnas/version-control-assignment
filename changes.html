How do I add my code to a repository?

If you want to add your code to an existing repository, you'll have to clone the directory to make your own copy. Get the URL from the github page you want to edit, and clone it in with this command.

Example Terminal
□ x
>/User/working_directory
$ git clone https://github.com/github_user/repository_name.git

Next, initialize your repository so that git recognizes it as a connected repository.

Example Terminal
□ x
>/User/working_directory
$ git init

The next step is to pull the most recent version of the code into your project.

Example Terminal
□ x
>/User/working_directory
$ git remote add upstream https://github.com/github_user/repository_name.git

Next, you should make your own branch before adding your own code to the project!

Example Terminal
□ x
>/User/working_directory
$ git branch my_branch_name

What's the next step?

The next step is to 'publish' your branch to the github repository.

To do this, first you need to 'stage' your code, which is the process of review before 'committing' your code. You can also choose to add single files with git add "file_name".

Example Terminal
□ x
>/User/working_directory
$ git add .

Now that you have staged your code, lets check the status of your code changes using the status command.

Example Terminal
□ x
>/User/working_directory
$ git status
